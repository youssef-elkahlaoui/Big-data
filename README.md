# Open Food Facts Dataset Instructions

## Dataset Download

To use this project, you'll need to download the Open Food Facts dataset separately:

1. Visit the official Open Food Facts data download page:
    https://world.openfoodfacts.org/data

2. Download the latest version of the dataset:
    - Look for "en.openfoodfacts.org.products.parquet" file
    - File size is approximately  4GB

3. No extraction needed as Parquet is a compressed columnar format

4. Place the Parquet file in your project's data directory

Note: The dataset is updated daily, so the file size and exact content may vary.

## Dataset Information
- Format: Parquet
- Size: ~ 4GB
- Contains: Food products information including ingredients, nutrition facts, and categories
