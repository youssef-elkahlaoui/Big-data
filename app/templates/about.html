About the Food Recommender System

This project uses the Open Food Facts dataset to build an intelligent food recommendation system.

Project Overview

The Food Recommender System is designed to help users discover food products that are similar to their favorites or find healthier alternatives with better nutritional profiles.

The system uses big data tools and machine learning techniques to analyze a large dataset of food products and provide personalized recommendations based on ingredients, nutritional information, and food categories.

Key Features

  • Content-Based Recommendations: Find similar products based on ingredients and product characteristics
  • Healthier Alternatives: Discover products with better Nutriscore ratings but similar ingredient profiles
  • Category Search: Browse products by category with filtering by nutritional quality
  • REST API: Access recommendations programmatically through a REST API

About Open Food Facts

This project uses data from the Open Food Facts database, a free, open, collaborative database of food products from around the world.

Open Food Facts is a non-profit project developed by thousands of volunteers from around the world.

Learn more about the dataset

Technical Implementation

Data Pipeline
  • Data Ingestion: Downloading and processing the OFF dataset
  • Preprocessing: Cleaning, normalization, and handling missing values
  • Feature Engineering: TF-IDF vectorization for ingredients and one-hot encoding for categories
  • Storage: Efficient storage using Parquet format
Technologies Used
  • Python: Core programming language
  • Pandas: Data manipulation and analysis
  • Scikit-learn: Machine learning and text vectorization
  • Flask: Web application framework
  • Bootstrap: Frontend design
  • NLTK: Natural language processing for text data
